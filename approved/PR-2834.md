## AI Summary
- **What it does**: Adds recovery logic for a known Ollama bug (TAURI-RUST-AZ, 208 Sentry events / 14d) where NaN float values in model output cause JSON serialization to fail with a 500 error. Instead of letting one bad input poison an entire embedding batch, the fix intercepts the NaN wire shape, re-issues each live input individually, and substitutes empty embeddings for inputs that still NaN-fail — preserving the length contract and Sentry observability.
- **Breaking risk**: Zero. Trait signature unchanged, return length preserved, empty-vec semantics pre-existing for blank inputs.
- **Security risk**: Zero. No new inputs trusted, no new network surface; NaN-matching is a read-only string check on the response body.
- **Bottom line**: Safe to merge.
