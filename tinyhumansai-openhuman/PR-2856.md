## AI Summary
- **What it does**: Adds a complete Phase 1+2 gameplay review feature — users import local keyframes, the core analyzes them via the existing screen intelligence engine, generates a session recap with highlights and clip candidates, produces platform-ready draft metadata for Twitch/Kick/YouTube, and supports follow-up Q&A on a session. Game-specific coaching presets with spoiler controls are also added. Supersedes PR #1677 with conflict resolution and CodeRabbit feedback applied.
- **Breaking risk**: Low. All additions are a new module (`gameplay_review`); only `src/core/all.rs` and `src/openhuman/mod.rs` are modified (wiring in the new controllers). No existing API signatures changed.
- **Security risk**: Low. Path traversal guard on session_id is present (`validate_session_id` checks for `/`, `\`, `..`). Images are base64-encoded locally, never uploaded to external endpoints. No hardcoded secrets. No auth bypass surface.
- **Bottom line**: Safe to merge once CI passes and the listGameplayPresets return type mismatch is fixed.
