## PR Description
Adds the Polymarket integration (read + trading) — the first of three venues called out in #1398 (Kalshi + Hyperliquid follow as separate PRs on the same umbrella).

- Read surface: market / event / orderbook / price discovery via Gamma + CLOB public APIs, plus authenticated reads (positions, balance, open orders, USDC.e allowance) once an EOA is configured.
- Trading surface: `place_order` + `cancel_order` against the Polymarket CLOB with EIP-712 typed-data signing on Polygon (chain_id 137).
- Approval gate: every write action requires `arguments.approved == true`. Stopgap until #1339's formal approval gate lands.
- Capability catalog split: `skills.polymarket_readonly` (browse) + `skills.polymarket_trading` (Beta, Action, destination = "Polymarket CLOB API").
