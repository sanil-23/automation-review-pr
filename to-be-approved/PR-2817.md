## AI Summary
- **What it does**: Adds `reasoning_content: Option<String>` to `ChatResponse` and `NativeMessage`, threads it through the OpenAI-compatible inference pipeline so DeepSeek's thinking-mode `reasoning_content` is preserved and replayed on follow-up requests. Without this, DeepSeek returns a 400 error on every multi-turn tool call with reasoning models. Fixes Sentry TAURI-RUST-4KB (31 events since v0.56.0).
- **Breaking risk**: Low. New field is `#[serde(skip_serializing_if = "Option::is_none")]` — zero wire change for non-reasoning providers. `build_native_assistant_history` signature change is internal; all call sites updated.
- **Security risk**: Zero. No user-facing input surfaces, no auth changes, no secrets.
- **Bottom line**: Safe to merge once CI is green.
