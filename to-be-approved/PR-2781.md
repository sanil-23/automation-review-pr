## AI Summary
- **What it does**: Adds `BackendApiError::Unauthorized { method, path }` to mirror the existing `MessageNotFound` pattern. Short-circuits `authed_json` on `status_code == 401` before `report_error` fires, logging at `info` with structured fields. Prevents ~12 events/2h Sentry noise from expected session-expiry 401s (OPENHUMAN-TAURI-4K8). 403 continues to report as intended.
- **Breaking risk**: Zero. Typed error is purely additive — callers that don't downcast see a generic anyhow error with a slightly different message string. No exported function signatures changed.
- **Security risk**: Zero. Suppresses noisy Sentry events for an expected, benign state; does not weaken auth, bypass validation, or expose data.
- **Bottom line**: Safe to merge.
