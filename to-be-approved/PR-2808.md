## AI Summary
- **What it does**: Adds `ExpectedErrorKind::EmptyProviderResponse` to `observability.rs` with a string classifier that catches the web-channel re-report of the agent harness's empty-provider-response error. When a provider/model returns an empty body, the typed `AgentError` gets flattened to a String at the native-bus boundary, so the typed `skips_sentry()` suppression from PR #2790 (TAURI-RUST-4JX) can't reach the web-channel re-report. This closes that second Sentry emit site (TAURI-RUST-4Z1) by demoting it to `tracing::warn!`.
- **Breaking risk**: Zero — purely additive: new enum variant, new predicate function, new dispatcher arm, new match arm.
- **Security risk**: Zero — pure in-process string classification. No new code paths, no network requests, no PII exposure beyond what was already in tracing.
- **Bottom line**: Safe to merge.
