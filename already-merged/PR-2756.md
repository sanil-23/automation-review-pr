## AI Summary
- **What it does**: Replaces the O(threads × messages) linear scan in `ConversationStore::search_cross_thread_messages` with a lazy, workspace-cached two-phase character n-gram inverted index. Adds multilingual normalization (NFKD decomposition, combining mark stripping, fold table for non-decomposing letters like ł/ß/ø) and CJK bigram support alongside Latin trigrams.
- **Breaking risk**: Zero. No public API changes; JSONL remains source of truth; score semantics (`matched_terms / total_terms`, `created_at` tiebreaker) preserved bit-for-bit with the legacy linear scan.
- **Security risk**: Zero. Pure in-memory computation on internal data; no new external inputs, no secrets, no injection surfaces.
- **Bottom line**: Safe to merge.
